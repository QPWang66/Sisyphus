# 西西弗斯 · Sisyphus

一个永远在推石头的桌面伴侣。

你敲键盘敲得越起劲，他推得越用力。石头永远到不了山顶。
你可度量的产出：**零**。（致敬阿尔贝·加缪）

他住在你桌面的一角 —— 崎岖山坡上一个发光的墨线小人，用书法般的笔画画成。
一圈一季，四季轮转。但重点是什么也没达成...

## 安装

从 [Releases](../../releases) 下载最新构建 —— `Sisyphus-macos-arm64.zip`
（Apple Silicon）或 `Sisyphus-windows-x64.exe`；或者从源码运行：

```sh
pip install pillow pynput
python3 -m sisyphus
```

**macOS 首次打开**（构建未经公证）：先双击让系统拦一次，然后
系统设置 → 隐私与安全性 → **仍要打开**。
或在终端清除隔离标记：`xattr -cr ~/Downloads/Sisyphus.app`。

**Windows** 可能弹出 SmartScreen：更多信息 → 仍要运行。

macOS：要让打字驱动推石头，需给 app 授予**输入监控**权限
（系统设置 → 隐私与安全性 → 输入监控），然后重启 app。
本地只保存每日击键的*数量*（`~/.sisyphus_stats.json`）—— 绝不记录按了哪些键。

## 互动

- **在任何地方打字** —— 他推得更猛。停下来，他就慢成疲惫的蠕行。
- **按住石头往上拖** —— 你可以帮他，最多帮到 99%。
  然后松手。你知道会发生什么。
- **点他一下** —— 他停下，直起身，安静地歇一会儿。
- **悬停在他身边** —— 今天的数字：`N 次击键 · 石头位移 0 米`。
- **光标停在他前进的路上** —— 他停下来等，一言不发。
- **匀速打字或狂乱爆发** —— 他的步态分得出区别。
- **拖其他地方**移动窗口。**右键**或 **Esc** 退出。

有些圈会带一点季节的痕迹 —— 草芽、萤火虫、一片落叶、稀疏的雪。
有些日子石头就是更大。偶尔他会滑倒。偶尔一只鸟落在石头上，
或一颗流星划过夜空，或远处的山脊上走过另一个西西弗斯。
极少数时候，石头差一点就成功了。深夜里，他有时会在石头旁
坐一会儿，然后重新开始。

**预览键**（先点一下窗口）：`1` 同行者 · `2` 流星 · `3` 鸟 ·
`4` 坐下 · `5` 滑倒 · `6` 差点成功 · `8` 沉重的石头。

## 自检

```sh
python3 -m sisyphus --check
```

## 结构

```
sisyphus/
  geometry.py   # 山坡、山顶、向量、样条
  sim.py        # 状态机（无窗口、可测试）
  render.py     # PIL 笔刷渲染：亮芯 + 模糊光晕
  inputs.py     # 键盘监听 + 每日计数
  app_mac.py    # AppKit 窗口（真透明）
  app_win.py    # Windows tk 色键窗口
  check.py      # 自检
```
